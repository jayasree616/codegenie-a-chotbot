# 🤖 CodeGenie – AI-Powered Personal Chatbot

CodeGenie is an intelligent, lightweight AI chatbot application designed to assist users with solving problems, generating responses, and managing conversations efficiently. Built using **Python**, **Streamlit**, and **LangChain**, this project demonstrates the integration of LLM-based conversational intelligence with a clean and user-friendly chat interface.

---

## 🚀 Features

- 💬 **Real-time Chat Interface** – Smooth, user-friendly chatbot UI built with Streamlit  
- 🧠 **LLM-powered Smart Responses** – Uses LangChain for text generation  
- 📁 **Chat History Storage** – Conversations stored in `chats.json`  
- 🔐 **Secret Key Handling** – Secure key management using `secrets.toml`  
- 📤 **File Upload Support** – Upload and process PDFs or images  
- 🗂 **Local Storage** – Saved uploads in `/uploads` directory  
- ⚡ **Fast & Lightweight** – Runs locally with minimal setup  

---

## 🛠️ Tech Stack

- **Python 3.10+**
- **Streamlit** – Web UI  
- **LangChain** – LLM response generation  
- **PyPDF2 / PDF Processing** (if used)  
- **JSON File Storage**
